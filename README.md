Multipad v1
A modular solution to make your workflow faster. Designed spesifically for video and code editing.
![IMG-0327.jpg](https://i.postimg.cc/P5D2fgtZ/IMG-0327.jpg)
How it Works
Modules (M1,etc) slot into the two slots of the Hub (H2) magnetically and automatically connect to the attached computer. The modules are 100% hot swap. Extra expandability can be achived by attaching Stands (S1) to the bottom of the H2. These act as a host for the H2 and modules on it. (meaning when using the stand module plug your computer into the stand instead of the hub)
![IMG-0337.jpg](https://i.postimg.cc/tJVVtHvg/IMG-0337.jpg)
Features
QMK/VIA Compatability (M1,M1_RGB)
Deej Compatability (https://github.com/omriharel/deej)
Windows + MacOS Compatability (except M2)
Up to 2 Hot Swappable modules on H2
Easy to expand usb interface in the 4 pin module interface
Magnetic stands (S1) to expand multipad capability
